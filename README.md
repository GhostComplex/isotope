# Isotope

A daemon-first AI agent platform with native Apple integration.

## Architecture

```
TypeScript monorepo:  @isotope/core → @isotope/agents → @isotope/daemon → @isotope/cli
Swift (separate):     isotope-apple-tools → isotope-app (macOS/iOS)
```
